Hyphens Text Layout

CSS utility classes for controlling hyphens word-break overflow-wrap text-overflow white-space line-clamp. Each class maps to a single CSS property for predictable, composable styling.

Hyphens

.hyphens-auto-tm This paragraph uses automatic hyphenation for elegant text wrapping. Words like antidisestablishmentarianism and supercalifragilisticexpialidocious will break gracefully at line endings.
.word-break-words-tm VeryLongWordWithoutSpacesThatNeedsToBreakBetweenCharacters applied here.
.text-overflow-ellipsis-tm This text is too long for its container and gets truncated with an ellipsis character at the end of the line.
.line-clamp-3-tm (in a column)
CSS line clamping is essential for creating uniform card layouts. It limits text to a specific number of lines and adds an ellipsis at the cut-off point. Perfect for article cards, product listings, and news feeds where visual consistency matters most.

White Space

.white-space-pre-wrap-tm This preserves whitespace and line breaks exactly as written in the source.
.truncate-2-tm
Product description that is way too long to fit on a single line so it gets clamped after exactly two lines of text and then the overflow is hidden automatically by the line-clamp utility.
